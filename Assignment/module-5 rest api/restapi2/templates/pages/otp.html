🔐 Google OAuth (Info)

Google Login Setup (Production): 1. Go to Google Cloud Console → APIs & Services → Credentials 2. Create OAuth 2.0 Client ID 3. Set Authorized redirect URI: http://localhost:8000/auth/google/callback/ 4. Install: pip install django-allauth 5. Add to settings.py: - GOOGLE_CLIENT_ID = 'your-client-id' - GOOGLE_CLIENT_SECRET = 'your-client-secret' 6. Configure django-allauth for Google provider ⚠️ This demo uses mock OTP. Set your Google credentials in .env for real OAuth.
