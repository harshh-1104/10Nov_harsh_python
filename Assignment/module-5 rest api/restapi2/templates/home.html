🔗 API Endpoints
  • GET /api/doctors/ — List all doctors (paginated)
  • POST /api/doctors/ — Create a new doctor
  • GET /api/doctors/<id>/ — Retrieve doctor details
  • PUT /api/doctors/<id>/ — Update a doctor
  • DEL /api/doctors/<id>/ — Delete a doctor
  • POST /api/register/ — Register new user
  • POST /api/login/ — Login & get token
  • GET /api/protected/ — Protected endpoint (requires token)
  • GET /api/joke/ — Random joke
  • GET /api/weather/?city=London — Weather data
  • GET /api/github/?username=octocat — GitHub repos
  • GET /api/country/?name=India — Country info
  • POST /api/otp/request/ — Request OTP
  • POST /api/otp/verify/ — Verify OTP
