A framework is a collection of pre-written code, libraries, tools, and conventions that provides a 
standard structure for developing software applications, where the framework controls the program 
flow and the developer writes code according to its rules.

=================orm================

object relational mapper

It is a tool that allows Python code to interact with a database without writing SQL queries.

| -----------------------| -----------------------|
| SQL    		| Django ORM  	|
| -----------------------| -----------------------|
| SELECT 		| filter()   		|
| INSERT 		| create()   	|
| UPDATE 	| save()     		|
| DELETE 	| delete()   	|

--------------------------------------------
sql---select
SELECT * FROM users WHERE username='harsh';

in python orm---
User.objects.filter(username='harsh')

---------------------------------------------

sql---insert
INSERT INTO auth_user (username,email)
VALUES ('harsh','h@gmail.com');

python---
User.objects.create(username='harsh', email='h@gmail.com')

---------------------------------------------

sql---update
UPDATE students
SET age = 23
WHERE id = 1;

python---
in model.py
class Student(models.Model):
    name = models.CharField(max_length=100)
    age = models.IntegerField()

in view
student = Student.objects.get(id=1)
student.age = 23
student.save()

--------------------------------------------------------------

sql---delete
DELETE FROM students
WHERE id = 2;

python---
student = Student.objects.get(id=2)
student.delete()

---------------------------------------------------------------

You can update directly without loading object.

sql---
UPDATE student
SET age = 24
WHERE id = 1;

python---
Student.objects.filter(id=1).update(age=24)
